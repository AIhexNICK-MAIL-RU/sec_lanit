# Lab 03: Nmap Network Scanning

## Выполненные задачи
1. Базовое сканирование портов
2. Определение версий сервисов
3. Обнаружение операционной системы
4. Сканирование с NSE скриптами
5. Анализ уязвимостей

## Использованные команды
- nmap localhost
- nmap -sV -p- localhost
- nmap -O localhost
- nmap --script=vuln localhost

## Результаты
Выявлены открытые порты и версии сервисов.
