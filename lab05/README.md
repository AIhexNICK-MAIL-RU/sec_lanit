# Lab 05: Docker Security and Container Analysis

## Выполненные задания

1. Установка Docker и buildkit
2. Анализ команд Docker build/run/save/load
3. Аудит безопасности Dockerfile
4. Создание .dockerignore
5. Замена скрипта на свой Python-файл
6. Сравнение хеш-сумм архивов
7. Доработка requirements.txt
8. Анализ слоёв и размера образа
9. Проверка пользователя в контейнере
10. Работа с Docker Hub
11. Изучение изоляции контейнеров (namespaces)
12. Проверка ресурсных лимитов
13. Создание Docker-сети
14. Запуск docker-compose
15. Очистка ресурсов

## Выводы

Лабораторная работа позволила на практике изучить ключевые аспекты безопасности контейнеризации, необходимые для DevSecOps-практик и обеспечения защиты контейнерных приложений.
