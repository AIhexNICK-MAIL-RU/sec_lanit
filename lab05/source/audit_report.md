# Аудит безопасности Dockerfile

## Проверка по чеклисту

| Проверка | Статус | Комментарий |
|----------|--------|-------------|
| Базовый образ официальный | Да | python:3.11-slim - официальный образ Python |
| Пиннинг версии (не latest) | Да | Версия 3.11-slim указана явно |
| Multi-stage build | Нет | Используется одноэтапная сборка |
| Инструкция USER | Нет | Контейнер запускается от root |
| Копирование только нужных файлов | Да | Копируется только hello.py |
| Наличие .dockerignore | Да | Файл создан, исключает ненужные файлы |
| Отсутствие секретов в ENV/ARG | Да | Нет секретов в Dockerfile |
| Пиннинг версий в requirements.txt | Нет | requirements.txt пустой |

## Рекомендации

1. Добавить multi-stage build для уменьшения размера образа
2. Добавить инструкцию USER для запуска от непривилегированного пользователя
3. Указать версии зависимостей в requirements.txt
4. Использовать .dockerignore для исключения чувствительных файлов
